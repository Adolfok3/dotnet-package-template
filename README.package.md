An awesome structured template for .NET packages.

## Requirements

- .NET 11.0 (preview) SDK or later

## Installation

```bash
dotnet add package DotnetPackageTemplate
```

Or via the Package Manager Console:

```powershell
Install-Package DotnetPackageTemplate
```
